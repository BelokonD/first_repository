Встановлення `npm install`

Запуск `npm start`

Програма виконує GET-запит до https://api.github.com
та виводить статус відповіді і отримані дані.

------------------------------------------------------

Встановленняя залежностей:

Перевірка встановлення node.js
```bash
node -v
npm -v
```

У папці проєкту виконати `npm install`

Команда встановить всі залежності, вказані у файлах package.json
